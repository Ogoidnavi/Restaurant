# Restaurant

This is a simple project to get more used to using Webpack in web development.

## Requirements

-   Node.js
-   NPM

## Installation

1. Clone the repository

```bash
git clone <URL>
```

2. Install dependencies:

```bash
npm install
```

# Development

To start the development server:

```bash
npm run dev
```

# Build

To bundle for production:

```bash
npm run build
```
